# Authentication_System

A Python-based authentication system providing simple user registration, login, and token-based authentication helpers.

## Features
- User registration and login flow
- Secure password hashing
- Token-based authentication (e.g., JWT support)
- Simple session or token validation utilities
- Minimal, easy-to-extend codebase (pure Python)

## Quickstart
1. Clone the repo:
   git clone https://github.com/Sagarrajak01/Authentication_System.git
2. Create a virtual environment and install dependencies:
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
3. Run the example or tests:
   python examples/run_example.py
